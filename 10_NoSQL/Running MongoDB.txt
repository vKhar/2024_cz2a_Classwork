MongoDB default database path c:\data\db

Start using cmd line:
Cmd at "C:\Program Files\MongoDB\Server\3.4\bin"
mongod
if you want to use another folder for your database 
mongod --dbpath=c:\MongoDB

Run as Windows Service
Run cmd as Administrator
mongod --install --dbpath=c:\MongoDB --logpath=c:\MongoDB\log.txt
net start MongoDB
net stop MongoDB
mongod --remove

check for port 27017

netstat -aon | findstr 27017
